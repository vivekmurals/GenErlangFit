# Generate Random Samples from Erlang + Exponential Mixture Distribution

Simulates random variates from a distribution defined as the sum of an
Erlang-distributed random variable and an independent Exponential random
variable.

## Usage

``` r
ErlangExpRnd(params, rows, cols = 1)
```

## Arguments

- params:

  Numeric vector of length 3 containing the parameters:

  - Erlang shape parameter `K` (integer)

  - Erlang rate parameter `lambda`

  - Exponential rate parameter `lambda`

- rows:

  Integer, number of rows for the output sample matrix.

- cols:

  Integer, number of columns for the output sample matrix (default 1).

## Value

Numeric matrix of random samples with dimensions `rows` Ã— `cols`.

## Details

Random samples are generated by independently sampling from the Erlang
and Exponential components and summing them. The output is reshaped as a
matrix.
